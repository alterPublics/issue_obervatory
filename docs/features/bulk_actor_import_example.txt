================================================================================
BEHAVIOR SEQUENCE
================================================================================

1. User clicks "Bulk Add" button
2. Single-actor form hides, bulk panel shows
3. User pastes/types actors in textarea
4. User clicks "Import Actors"
5. Frontend validates line-by-line
   - If error: Show error message with line number, keep textarea content
   - If valid: Show spinner, disable button, POST to backend
6. Backend validates and processes
   - Creates/links canonical Actor records
   - Skips duplicates
   - Returns counts
7. Success: Show message "Added X actors, skipped Y duplicates"
8. After 800ms: Page reloads to show new actors
9. User sees all actors in the Actor List below
