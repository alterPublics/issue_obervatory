Zeeschuimer Integration

Browser extension for manual data capture

Zeeschuimer is a browser extension for capturing social media data that cannot be accessed via API. Supported platforms:

  • LinkedIn
  • Twitter/X (supplement to API)
  • Instagram
  • TikTok
  • Threads
Install Zeeschuimer
{# File format requirements #}

File Format Requirements

{# CSV format #}

CSV

Required columns:

  • platform (required)
  • url (optional)
  • title (optional)
  • text (optional)
  • published_at (optional)
  • author_display_name (optional)
{# NDJSON format #}

NDJSON

One JSON object per line with:

  • platform (required)
  • Any platform-specific fields

Platform is auto-detected for Zeeschuimer files.

Note: Files larger than 50 MB or with more than 10% row errors will be rejected. Large datasets should be split into multiple files.
